# Review huidige migratielijn

Deze map bundelt de eerste reviewronde voor de nieuwe `next`-omgeving.

Doel:

- opmerkingen over inhoud, structuur en vormgeving niet laten versnipperen;
- de eerste gemigreerde productlijn gericht verbeteren;
- en reviewwerk scheiden van de technische migratie.

Huidige scope:

- `Objectenhandboek` landing
- `Watersysteem`
- `Watergangen`
- `Watergangsectie`
- `Intersectie`

Werkwijze:

1. Verzamel opmerkingen in `01-Objectenhandboek-reviewlog.md`
2. Cluster ze per thema
3. Zet goedgekeurde aanpassingen om naar concrete backlog-items
4. Werk ze daarna in kleine, controleerbare slices uit

Reviewthema's:

- inhoudelijke juistheid
- hiërarchie en navigatie
- consistentie tussen pagina's
- visuele harmonisatie
- beheerbaarheid van content en componenten
