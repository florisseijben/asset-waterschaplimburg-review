Wat deze pagina test

  • Kan gewone JSON via fetch() geladen worden?
  • Kan GeoJSON via fetch() geladen worden?
  • Kan inline JSON-LD gelezen en getoond worden?
  • Blijft de pagina bruikbaar als lokale data-opvraag wordt geblokkeerd?

Interpretatie

Als alleen JSON-LD en fallback werken, dan is route B nog steeds mogelijk, maar dan moeten we HTML-first bouwen en data alleen als optionele verrijking gebruiken.
